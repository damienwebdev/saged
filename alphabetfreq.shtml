Notes

Alphabet frequency is useful in deciphering basic encryptions for which each letter corresponds to a different encrypted letter. Using the table below, we can guess that the most frequently appearing letter in our encrypted string corresponds to the letter 'E', next 'T', and so on. Note that this strategy takes some detective work and will not work if employed on a short encrypted string. It is also useful to examine common n-grams.
