Purpose

The purpose of this site is to be used as a tool for advanced mathematical computations.
As many computations in Number Theory and Cryptography require immense calculations, this site provides a simple way for any amateur mathematician to do individual calculations without having to learn how to code.

Sage

To allow for large number computations, this website implements the SAGE programming language.
Sage uses many of the most advanced methodologies, algorithms, and computational engines to provide near unfettered computation capabilities.
Sage implements calculators like Pari to do many of its calculations.
